Uniform integrability, Equi-integrability and Dunford-Pettis Theorem

Before stating Dunford-Pettis Theorem, we state some definitions.

Definition (Equi-integrability). For Ω⊂Rn\Omega\subset \mathbb{R}^nΩ⊂Rn, a family of integrable functions U⊂L1(Ω)\mathcal{U}\subset L^1(\Omega)U⊂L1(Ω) is equi-integrable if the following two conditions hold

  • (Tight) The set U\mathcal{U}U is tight, i.e., for every ε>0\varepsilon>0ε>0 there exists a measurable set AAA with finite measure such that (trivially true if Ω\OmegaΩ is of finite measure) sup⁡u∈U∫Ω−A∣u(x)∣dx<ε. \sup_{u\in \mathcal{U}} \int_{\Omega - A} |u(x)|dx < \varepsilon. u∈Usup​∫Ω−A​∣u(x)∣dx<ε.
  • (Uniform integrable) For every ε>0\varepsilon > 0ε>0 there exists δ>0\delta > 0δ>0 such that if EEE is a measurable set with ∣E∣<δ|E| < \delta∣E∣<δ then sup⁡u∈U∫E∣u(x)∣dx<ε. \sup_{u\in \mathcal{U}} \int_E |u(x)|dx < \varepsilon. u∈Usup​∫E​∣u(x)∣dx<ε.

Now we state the Dunford-Pettis theorem that characterizes weak convergence in σ(L1,L∞)\sigma(L^1,L^\infty)σ(L1,L∞).

Dunford-Pettis Theorem. For Ω⊂Rn\Omega\subset \mathbb{R}^nΩ⊂Rn, a sequence (un)⊂L1(Ω)(u_n)\subset L^1(\Omega)(un​)⊂L1(Ω) is convergent weakly in σ(L1,L∞)\sigma(L^1,L^\infty)σ(L1,L∞) if and only if

  • The sequence (un)(u_n)(un​) is equi-integrable.
  • The sequence (un)(u_n)(un​) is uniformly bounded in L1L^1L1, i.e., sup⁡n∥un∥L1(Ω)≤C\sup_n \Vert u_n\Vert_{L^1(\Omega)} \leq Csupn​∥un​∥L1(Ω)​≤C.
